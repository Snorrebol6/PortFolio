Bon Appmobile

Project summary

Bon Appmobile is a VR cooking simulator. The goal of the game is to bake pizzas and deliver them to houses. You drive around in a food truck in a large open world, where you can decide where to go.

My work/what I learned

I, together with Mathijs(a Student from Grafisch Lyceum Utrecht), created the sauce gun. First, it checks if you have picked it up and with which hand. The controls activate based on the hand you're holding it with. When you press the shooting button, the particle system activates. If the particle hits something, it triggers an event. When that event is called, it checks with a sphere cast whether it's a pizza. If it is, a function is called to display the sauce on the pizza.

We also got around to making an ordering system where my primary task was to create the delivery point with randomized ingredients for the pizza. Sadly, we never got the time to finish and implement it.
